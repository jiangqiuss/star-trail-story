# 🌟 星轨物语：记忆碎片

一款基于星空主题的抽卡文字冒险游戏，收集卡牌、解锁故事、探索记忆碎片。

## 🎮 在线游玩

👉 [点击这里开始游戏](https://jiangqiuss.github.io/star-trail-story/)

## ✨ 游戏特色

### 🎴 抽卡系统
- **5个稀有度**：N → R → SR → SSR → UR
- **30+ 张独特卡牌**，每张都有专属故事
- **10连抽** 保底 SR
- **抽卡动画** 沉浸式体验

### 📖 故事系统
- 每张卡牌都承载着一段记忆碎片
- 收集卡牌解锁完整故事线
- 星空背景下的沉浸式叙事

### 🎨 视觉设计
- 动态星空背景
- 流光粒子特效
- 响应式设计，支持移动端

## 🎯 游戏玩法

1. **抽卡** - 使用星尘抽取卡牌
2. **收集** - 收集不同稀有度的卡牌
3. **阅读** - 点击卡牌阅读专属故事
4. **探索** - 解锁更多记忆碎片

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 星空动画
- 无任何外部依赖
- localStorage 本地存储

## 📦 部署

直接将 index.html 托管到任意静态服务器即可。推荐使用 GitHub Pages：

1. Fork 本仓库
2. 进入 Settings → Pages
3. Source 选择 main 分支
